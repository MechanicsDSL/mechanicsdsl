# Community Contributions

Third-party integrations, community plugins, and contributed tools.

## Structure

```
contrib/
├── plugins/          # Community-developed plugins
├── integrations/     # Third-party system integrations
├── tools/            # Community tools and utilities
└── examples/         # Community-contributed examples
```

## How to Contribute

1. Fork the repository
2. Create a new directory under the appropriate category
3. Include a `README.md` describing your contribution
4. Submit a pull request

## Guidelines

- All contributions must include tests
- Follow the project's coding style
- Include documentation
- License must be MIT-compatible
