# Julia Pendulum Demo

Simple pendulum simulation using Julia's DifferentialEquations.jl.

## Files

- `pendulum.jl` - Generated Julia script

## Requirements

```julia
using Pkg
Pkg.add(["DifferentialEquations", "Plots", "CSV", "DataFrames"])
```

## Run

```bash
julia pendulum.jl
```

## Visualize

The script automatically generates `simple_pendulum_results.csv` and a plot.
