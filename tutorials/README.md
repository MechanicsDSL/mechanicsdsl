# MechanicsDSL Tutorials

Interactive Jupyter notebooks for learning MechanicsDSL.

## Tutorials

| # | Tutorial | Topics |
|---|----------|--------|
| 1 | [Getting Started](01_getting_started.ipynb) | DSL basics, simple pendulum, export |
| 2 | [Double Pendulum](02_double_pendulum.ipynb) | Chaos, sensitivity, phase space |
| 3 | [Parameter Estimation](03_parameter_estimation.ipynb) | Inverse problems, Sobol analysis |

## Running

```bash
pip install mechanicsdsl-core jupyter
cd tutorials
jupyter notebook
```
