# Maintenance Mode Notice

**Effective Date:** January 30, 2026

---

## Status

MechanicsDSL has entered **maintenance mode**. This means:

### ✅ What We Will Continue

- **Security patches** — Critical vulnerabilities will be addressed promptly
- **Dependency updates** — Keeping compatible with Python 3.9+ and core dependencies
- **Bug fixes** — Critical bugs affecting core functionality
- **Documentation updates** — Correcting errors and clarifying existing features

### ⏸️ What Is Paused

- **New feature development** — No new physics domains, code generators, or major features
- **Performance optimizations** — Unless required for compatibility
- **New platform support** — No additional target platforms

### ❌ What Is Not Planned

- Major architectural changes
- Breaking API changes
- New integrations

## Contributing During Maintenance Mode

We still welcome contributions! Priority is given to:

1. **Security fixes** — Always accepted
2. **Bug fixes** — With regression tests
3. **Documentation improvements** — Clarifications and corrections
4. **Test coverage** — Additional tests for existing functionality

Lower priority:
- New features (may not be reviewed promptly)
- Refactoring (unless addressing specific bugs)
