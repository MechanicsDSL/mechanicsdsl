# MechanicsDSL Assets

Brand assets and visual resources.

## Logo

The MechanicsDSL logo is available in multiple formats:

| File | Format | Usage |
|------|--------|-------|
| `logo.svg` | Vector | Web, documentation, presentations |
| `logo-dark.svg` | Vector | Dark mode backgrounds |
| `icon.png` | 512x512 PNG | App icons, favicons |
| `banner.png` | 1200x630 PNG | Social media, GitHub |

## Color Palette

| Name | Hex | Usage |
|------|-----|-------|
| Primary | `#2563EB` | Links, buttons, accents |
| Secondary | `#7C3AED` | Gradients, highlights |
| Dark | `#1E293B` | Text, headers |
| Light | `#F8FAFC` | Backgrounds |
| Success | `#10B981` | Success states |
| Warning | `#F59E0B` | Warnings |
| Error | `#EF4444` | Errors |

## Typography

- **Headings**: Inter or SF Pro Display
- **Body**: Inter or system-ui
- **Code**: JetBrains Mono or SF Mono

## Usage Guidelines

1. Maintain minimum clear space around the logo
2. Do not stretch or distort
3. Use appropriate color version for background
4. Minimum size: 32px height
