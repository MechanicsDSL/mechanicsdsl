## Contributing a Translation

We welcome translations in any language! To contribute:

1. **Fork the repository**
2. **Create a new folder** with the appropriate language code (e.g., `fr/` for French, `ko/` for Korean)
3. **Translate the README.md** as a starting point
4. **Submit a pull request**

### Language Codes

Use [ISO 639-1](https://en.wikipedia.org/wiki/List_of_ISO_639-1_codes) two-letter codes:
